Lovira es una app Android pensada para mantener una conexión cercana con una persona especial: gestos rápidos, estado en tiempo real, autenticación segura y notificaciones push.

Esta vista recrea el flujo principal para el portafolio sin depender de un emulador externo.

Notificaciones push

Integración con OneSignal para enviar avisos afectivos en tiempo real.

Base de datos en tiempo real

Uso de Firebase Firestore para sincronizar estados y mensajes entre usuarios.

Autenticación

Flujo de registro e inicio de sesión con Firebase Auth.

Tecnologías utilizadas

Kotlin Android SDK Firebase OneSignal Material Design
